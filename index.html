Rules:

Boxes will be blinked randomly by the computer.

Only a single box will be blinked at one level.

You have to remember the pattern of blinking of boxes from the first level to the current level.

At every level you have to click the boxes in same pattern in which they blinked since first level.

On clicking correct pattern you will move to next level.

If clicked boxes in wrong pattern then game will get over.

Press any Key to Start

Highest Score: 0
